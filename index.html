Html

HTML: Acts as the skeleton of web pages, utilizing tags to define elements such as headings, paragraphs, links, and images. It forms the structure that CSS and JavaScript interact with to create a complete web experience.

Css

CSS: Allows for the customization and styling of HTML elements, controlling layout, colors, fonts, and overall presentation. It separates design from content, promoting consistency and visual appeal across a website.

JavaScript

JavaScript: Enables interactive elements like forms, animations, and responsive features. It is a versatile language that can modify content, respond to user actions, and communicate with servers to update web pages dynamically.

Jquery

jQuery: Simplifies JavaScript programming by providing a library of pre-written JavaScript functions, enabling easier DOM manipulation and event handling.

React

React: A JavaScript library for building user interfaces, focusing on component-based architecture for creating efficient and reusable UI elements.

Font-Awsome

Font Awesome: A library offering a wide array of scalable vector icons that can be easily customized and integrated into web projects.
